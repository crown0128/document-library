* **难度**：简单
* **涉及知识**：位运算
* **题目地址**：https://leetcode-cn.com/problems/hamming-distance/
* **题目内容**：

```
两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

给出两个整数 x 和 y，计算它们之间的汉明距离。

注意：
0 ≤ x, y < 231.

示例:

输入: x = 1, y = 4

输出: 2

解释:
1   (0 0 0 1)
4   (0 1 0 0)
       ↑   ↑

上面的箭头指出了对应二进制位不同的位置。
```

> [返回目录](#chapter-one)

> index.js

```js
/**
 * @param {number} x
 * @param {number} y
 * @return {number}
 */
const hammingDistance = (x, y) => {
  let cnt = 0;
  while (x != 0 || y != 0) {
    if ((x & 1) != (y & 1)) {
      cnt++;
    }
    x >>= 1;
    y >>= 1;
  }
  return cnt;
};

console.log(hammingDistance(1, 4));
```

`node index.js` 返回：

```js
2
```
