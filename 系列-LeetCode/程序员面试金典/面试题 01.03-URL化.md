## <a name="chapter-three" id="chapter-three"></a>三 解题思路

> [返回目录](#chapter-one)

花费 03分48秒，提出 2 种题解：

1. 暴力
2. 正则

---

> 02:12 解法 1：暴力

```js
const replaceSpaces = (S, length) => {
  let str = '';
  for (let i = 0; i < length; i++) {
    if (S[i] !== ' ') {
      str += S[i];
    } else {
      str += '%20';
    }
  }
  return str;
};
```

---

> 03:48 解法 2：正则

```js
const replaceSpaces = (S, length) => S.slice(0, length).replace(/\s/g, '%20');
```
