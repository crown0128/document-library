## 推荐学习方法
&emsp;其实来说，真要学习的话，其实不应该跟着 jsliang 一起学习~ 因为 jsliang 并没有表面上的那么勤奋。  
&emsp;如果你是技术胖群里的小伙伴，平时可以看看邵哥的文章，看看胖哥的视频。jsliang 的话，只是想起学点东西，觉得好奇，就去学了。  
&emsp;然后，建议你们有机会就自己配个显示器，也就5/600，用起来也舒服，jsliang 就是一台手提（ 16G 6核 12线程 7200RMB ） + 一个显示器 （ 21.5寸 1080P 650RMB )。学前端来说，这个价还是用得起的，而且自己的配置，自己用得舒服最好。  
&emsp;最后，祝小伙伴们找到自己的节奏，工资步步高升。
