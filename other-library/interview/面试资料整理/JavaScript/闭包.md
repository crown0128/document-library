### <a name="chapter-three-one" id="chapter-three-one"></a>3.1 题目 1：求打印结果

> [返回目录](#chapter-one)
  
```js
function test() {
  var n = 4399;

  function add() {
    n++;
    console.log(n);
  }

  return {
    n,
    add
  };
};

var result = test();
var result2 = test();

result.add(); // 输出啥
result.add(); // 输出啥

console.log(result.n); // 输出啥

result2.add(); // 输出啥
```

选择：

* A：4400 4401 4399 4400
* B：4400 4401 4401 4402
* C：4400 4400 4399 4400
* D：4400 4401 4399 4402
* E：4400 4401 4401 4400

---

答案：A
