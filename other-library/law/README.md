出来混，小心处事是非常必要的。

这年头，随便干点事可能律师函就来了。

在 **jsliang** 短暂的人生中，就曾见过：

* 禁止向其他公司透露本司机密，违者罚款 100 万。
* 酌情支付额外的薪酬。
* ……

那么，这些条约有效吗？我们该如何防止自己陷入 *精明人* 的套路中呢？

所以，学习一下相关的知识吧！
