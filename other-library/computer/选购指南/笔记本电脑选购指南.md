### <a name="chapter-three-three" id="chapter-three-three">3.3 参数 - 处理器</a>

> [返回目录](#chapter-one)

处理器分两个方面讲解：

1. 品牌
2. 代数

* 品牌

品牌是电脑中最重要的部件。处理器目前有两大厂商：AMD 和 Intel，相对应的即是 APU 和 CPU，建议购买 Intel 的 CPU 处理器。

* 代数

现在 4000 元以上的电脑基本上有 8 代处理器了，像 i5 8250u，i7 8550u。

在这里说明下参数：i5 后面的第一个数字 8 即代表第 8 代，后面的 u 代表低频，一般来说带 u 的处理器不适合打游戏，LOL 这种还是可以的，带 u 型号的 CPU 基本是超薄本，好处是轻便便于携带，适合办公。当然，散热方面要注意。

如果你需要玩游戏，那么建议 PUBG，CPU 最少也要 i7 8750h，而且还有好的显卡（最好 1060 以上）。

买电脑不建议买 4000 元以下的，目前至少也要 i5 8 代的处理器（即 i5 8250u 之类的）。

### <a name="chapter-three-four" id="chapter-three-four">3.4 参数 - 主板</a>

> [返回目录](#chapter-one)

笔记本主板是笔记本电脑中各种硬件传输数据、信息的立交桥。

笔记本主板整合了显卡、内存、CPU 等各种硬件，使其相互独立而又有机地结合在一起，融洽地维持电脑正常运行。

笔记本主板是笔记本电脑上的核心配件，不同机型的机器用的主板也有所不同，甚至是同一个型号的机器也可能有区别。

目前品牌有：惠普、戴尔、联想、华硕、三星、神州、苹果……等

主板由中央处理器（CPU）和显卡组成。

### <a name="chapter-three-five" id="chapter-three-five">3.5 参数 - 内存</a>

> [返回目录](#chapter-one)

牌子：内存常见的牌子有三星、镁光、金士顿。

代数：目前笔记本都以 ddr4 即 2133hz、2400hz 或者 2400hz 以上频率的。如果还是 drr3 即 1600hz 和 1600hz 以下的，建议不要购买，买了就是后悔。

容量：office、看电影、微信、QQ 等同时开最少需要 8G，而且 8G 基本是主流电脑的配置了。

如果你玩游戏，如果你还是程序猿，那么建议 16G。

如果你不玩游戏，那么市面上普遍的 8G 就够了。

### <a name="chapter-three-six" id="chapter-three-six">3.6 参数 - 硬盘</a>

> [返回目录](#chapter-one)

影响系统运行快慢的主要硬件，有机械硬盘和固态硬派两种，固态硬盘市场上有 SATA 接口和 PCIE 接口两种。

SATA 和传统的硬盘接口一样，但是速度比机械硬盘快很多，在速度上：PCIE > SATA > 机械硬盘。

目前市场上大小以 256G 为主（实际也就 230+G），足够使用了。

目前硬盘配置上，分为两种形式：

1. 固态硬盘。不能自己家机械硬盘。
2. 固态硬盘 + 机械硬盘混合的。一般是 256G SSD + 1T 机械硬盘。

### <a name="chapter-three-seven" id="chapter-three-seven">3.7 参数 - 显卡</a>

> [返回目录](#chapter-one)

显卡，即图形处理，市场上超薄本常见的有 MX150 和它的升级版本 MX250。

如果你需要使用 Adobe系列（PS、Pr、Ae……）的工具，建议搭配上 MX150 和 MX250 显卡，当然 8 代处理器自带的核显也够普通的 PS 之类的 Adobe 产品使用了，只是会增加 CPU 负担，减少电脑寿命。

显存：和内存差不多，缓存作用，游戏显卡的话，目前来说建议 1660，即 1060 升级版。

品牌：一线品牌华硕、七彩虹、微星等。

### <a name="chapter-three-eight" id="chapter-three-eight">3.8 参数 - 手提显示器</a>

> [返回目录](#chapter-one)

分辨率：没有 1080P 的不要选。

色域：不是绘图专业人士，普通的就可以了，默认是 45% 色域的，72% 色域的要贵许多，但是 RGB 色彩显示方面好。

材质：主要有 IPS 屏幕和 LED，建议 IPS 屏幕。

大小：主流是 14~15 寸，看个人喜好，觉得小可以买个显示器扩展。

### <a name="chapter-three-night" id="chapter-three-night">3.9 参数 - 单独显示器</a>

> [返回目录](#chapter-one)

常见牌子有：AOC、三星、惠普、DELL 等，但是性价比高的建议 AOC。

建议 24~27 存为佳，曲面平面看个人喜好。

## <a name="chapter-four" id="chapter-four">四 选购攻略</a>

> [返回目录](#chapter-one)

看到这里，恭喜小伙伴们初步了解笔记本电脑的参数配置。

那么，咱们在选购电脑的时候，需要注意什么呢？

Office 电脑配置推荐（4k ~ 6k）：

* CPU：i5 8250u < i5 8265u < i7 8550u < i7 8565u
* 内存：8G ddr4
* 固态：SATA < PCIE 256G
* 显卡：集显 < MX150 < MX250，显存 2G < 4G
* 屏幕：14/15 存，IPS 材质、1080P 分辨率

游戏本基本配置：

* CPU：i5-9300h < i7 9750h
* 内存：8G drr4
* 固态：512G PCIE
* 屏幕：1080P，72% 色域 144hz
* 显卡：GTX 1650 或以上
