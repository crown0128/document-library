前端学习路线 - 2020
===

> Create by **jsliang** on **2020-2-11 09:26:00**  
> Recently revised in **2020-2-17 18:56:11**

大厂是围城：

* 外面的人想进去。围城里面的人喷进度、喷晋升、喷业务、喷管理，围城里面仿佛什么都不好。
* 里面的人想出来。围城外面的人在祈福、在羡慕、在感叹，感觉围城里面什么都很棒。

围城围住的向来都是你自己难以迈步的心，有想法就去追，有差距就去学习。

1. 造三个 npm 包
2. 操作 Linux 系统
3. Ajax 详细了解与个人封装
4. 直播系统（即时通讯）
5. 算法与数据结构
6. 设计模式
7. 做一个个人时间投资计划
