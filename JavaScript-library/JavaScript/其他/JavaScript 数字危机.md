有一个数字：`1323.9547499999996`。

按照正常人的思维，将其四舍五入，保留两位小数，结果是不是应该为：`1323.96`

但是，当我引用：

```js
import NP from 'number-precision';
import _ from 'lodash';
```

使用：

```js
NP.round(1323.9547499999996, 2)
_.round(1323.9547499999996, 2)
```

得到的结果都为 `1323.95`，这就非常尴尬了。

网上找了个方法：

```js
const roundDecimal = (val, precision) => {
  return Math.round(Math.round(val * Math.pow(10, (precision || 0) + 1)) / 10) / Math.pow(10, (precision || 0));
}
roundDecimal(1323.9547499999996, 2);
```

居然得到了正确的结果，无奈了。
