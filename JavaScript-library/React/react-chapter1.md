&emsp;工欲善其事，必先利其器。  
&emsp;在这里，我们将进行 React 的基础环境的搭建，从而大大增加我们的开发进度。

### 1.1 搭建 React 脚手架
1. 安装 React 脚手架：`npm i create-react-app -g`
2. 安装 React 项目：`create-react-app todolist`
3. 运行 React 项目：`npm run start`

> 步骤 1 结果：返回create-react-app 版本：1.5.2
